1、说明

box-shadow: X轴偏移量 Y轴偏移量 [阴影模糊半径] [阴影半径] [阴影颜色] [投影方式]
添加多个阴影，用逗号隔开

2、外阴影

```css
.outer{
    width:100px;
    height:100px;
    background: steelblue;
    box-shadow: 10px 10px 20px 5px black ;
}
```
第三个参数大一点，第4个参数小一点，会有比较好的模糊效果
